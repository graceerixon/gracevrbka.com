01 The work currently in production
Flagship 2024 → now

Superb Shifts. A two-sided healthcare staffing platform serving 15,010 users and 158 senior living facilities.

Own all infrastructure, frontend, backend, security, and new feature development. Joined as a full-stack engineer, promoted to CTO 18 months in, sole engineer since October 2025.

A

UI and onboarding

Redesigned and refactored the UI across three user types (facilities, healthcare workers, admins) and iteratively overhauled the onboarding flow based on real user friction. Migrated all views to a consistent component library for visual and functional parity.

B

Email and comms

Designed an email marketing system from scratch: SendGrid on the backend, MJML responsive templates, a React Email Editor drag-and-drop builder on the frontend, and audience segmentation. Integrated a multi-channel communication layer spanning Stream Chat, Stream Video, Firebase push notifications, Twilio SMS, and SendGrid.

C

Admin and ops tooling

Grew the admin tooling to support 15,000+ user records at scale: worker search and filtering, shift metrics dashboards, event logging, user impersonation for debugging, bulk messaging, and PostHog analytics with feature flags. Optimized data-heavy pages with virtualized rendering and lightweight API endpoints.

D

Subcontractor portal

Identified an opportunity to bring third-party staffing agencies onto the platform as a new revenue channel. Built the portal end-to-end: full NestJS backend module, role-based access control, and contract management.
